# PDF to Markdown Converter for LLMs 🚀

A simple and efficient Python tool that converts PDF documents into clean Markdown files. This is specifically optimized for feeding data into Large Language Models (LLMs) or AI contexts.

## 🛠️ Technologies
* **Python 3.x**
* **PyMuPDF4LLM**: High-performance PDF to Markdown conversion.
* **WSL2 / Linux Terminal**

## 📂 Project Structure
* `source_docs/`: Place your input PDF files here.
* `converted_docs/`: The script will save the generated Markdown files here.
* `main.py`: The core conversion script.
